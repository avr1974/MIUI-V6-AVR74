说明


1. 开发说明


在开发应用程序时，仅需关注文档中所提供的接口函数而不用了解具体实现。


2. 支持的平台


支持Android 2.0以上的系统
 


3. 开发包组成


Document
 ------ 《语点套件3.0开发文档.chm》


JarLib
 ------ SpeechServiceLibUtil.jar 封装了连接服务、共享资源等功能的Jar包


SpeechClient
 ------ 标准应用。


4. 在新建工程中使用语点引擎


1. 安装SpeechService.apk。


2. 将SpeechClient工程下的SpeechServiceLibUtil.jar包增加到新工程中。


3. 在AndroidManifest.xml中声明必要的权限（参见SpeechClient的权限）。
