套件3.0开发流程简要说明

1.构建SpeechServiceUtil对象,传入上下文、服务绑定回调以及初始化参数：

ISpeechInitListener mListerner = new ISpeechInitListener(){…};//创建服务绑定监听回调

SpeechServiceUtil mService = new SpeechServiceUtil(context,mListerner，intent);

intent为Intent对象，根据使用需要传入对应的参数值。

2.在服务绑定完成回调mListerner.OnInit()中进行各引擎的初始化，可以根据客户端需要有选择的进行引擎初始化：

RecognitionListener.Stub mRecListener = new RecognitionListener.Stub(){…};//创建识别引擎回调接口

mService.initRecognitionEngine(mRecListener,intent)//初始化识别引擎

SynthesizerListener.Stub mSynListener = new SynthesizerListener.Stub(){…};//创建合成引擎回调接口

mService.initSynthesizerEngine(mSynListener,intent)//初始化合成引擎

VerifierListener.Stub mVerListener = new VerfierListener.Stub(){…};//创建声纹引擎回调接口

mService.initSpeakerVerifier(mVerListener,intent)//初始化声纹引擎

intent为Intent对象，根据使用需要传入对应的参数值。

3.注意引擎初始化以后是否返回OnInit回调，如果返回，说明引擎初始化成功，可以调用引擎其他的功能接口了；如果没有返回OnInit，说明引擎初始化失败，需要查看返回的错误码，确认初始化参数是否有错误。（注：合成引擎的OnInit在每次开始speak()的时候回调。）

4.引擎初始化成功以后，就可以调用引擎对应的语音能力接口，以识别为例：

//开始识别

mService.startRecognize(intent);//intent为Intent对象，根据识别需要传入对应的参数值；

在传入正确的参数后，识别引擎会依次回调：

OnRecordStart();//开始录音

OnSpeechStart();//VAD引擎检测的音频前端点

OnSpeechEnd();//VAD引擎检测的音频后端点

OnRecordEnd();//停止录音

OnResult();//返回识别结果,RecognizerResult对象。

完成识别过程。

5.引擎销毁，当希望退出套件3.0语音服务时调用:

mService.destroy();
